# H2 答题导出脚本

## 目标
- 从 H2 `.mv.db` 文件导出用户答题内容
- 按 data.json 的章节/题目顺序生成目录结构
- 将 HTML 中的 base64 图片提取为文件并引用

## 实现
- 新增脚本：`ai_script/export_answers_from_h2.py`
- 使用 `data.json` 作为用户与题目顺序来源（排除 admin）
- 目录命名安全替换非法字符
- 输出路径：`data/answer_YYMMDD/<username>/<chapterIndex>_<questionIndex>_<questionTitle>/`
- 图片文件：`image_001.ext`…，HTML 以相对路径引用

## 依赖
- `jaydebeapi`
- `JPype1`

## 验证
- 使用 `data/pocdb_stress.mv.db` 成功导出
- 生成路径：`data/answer_260128`
- 示例 HTML 中已替换为相对图片引用
